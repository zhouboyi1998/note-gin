### ⌛ Start

#### Project configuration

* 1：配置 `Global GOPATH` & `Project GOPATH`
* 2：配置 `Environment`
    * `GOPROXY=https://goproxy.cn,direct`
    * `GOFLAGS=-buildvcs=false`

---

#### Install dependencies

* Run the command in the root path of the project：`go mod tidy`
